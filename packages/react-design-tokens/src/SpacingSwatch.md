Spacing swatch renders a bar with a width of the provided `value` and its content as a text

```jsx harmony
<SpacingSwatch value={"1rem"} />
```

You can override default styles using `css` prop:

```jsx harmony
<SpacingSwatch
  value={"100px"}
  css={{
    color: "white",
    background: "cornflowerblue"
  }}
/>
```
